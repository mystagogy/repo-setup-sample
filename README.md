# repo-setup-sample

이 프로젝트는 Github에서 Repository를 사용할 때 관리해야 할 것들을 연습하기 위한 Repository입니다.

[See Demo] (https://github.com/mystagogy/repo-setup-sample.git)

## Prerequisite

- node.js: ^11.0
- java: ^8

## How to Start

```shell
$git clone {repo-address}
$cd repo-setup-sample
$npm install
$npm start
```

## Features

- Blog
- Reply
- My Page
- Donate with bitcoin
